# PrivoxPTT — Web Client

PWA приложение для абонентов и диспетчеров. Открывается в браузере, устанавливается как приложение на телефон.

## Быстрый старт

```bash
cd web
cp .env.example .env
npm install
npm run dev
```

Открыть: http://localhost:5173

## Доступ по роли после входа

| Роль | URL | Описание |
|------|-----|----------|
| `USER` | `/radio` | Экран рации: PTT кнопка, каналы, абоненты |
| `DISPATCHER` | `/dispatcher` | Пульт диспетчера + карта |
| `ADMIN` | `/admin` | Управление пользователями, группами |
| `SUPERADMIN` | `/admin` | + управление организациями |

## Клавиши

- **Пробел** — удержать для передачи PTT (десктоп)
- **Тач** — удержать кнопку PTT (телефон)

## PWA установка

На мобильном устройстве: в браузере → "Добавить на главный экран". Работает офлайн (кешированные ресурсы).

## Сборка для продакшена

```bash
npm run build
# Файлы в dist/ — статический хостинг или Nginx
```
